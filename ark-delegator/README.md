# ark-delegator

REST client for Arkade delegator services.

This crate provides typed request/response helpers for interacting with Arkade delegator APIs used by the Arkade Rust SDK.

## Install

```toml
[dependencies]
ark-delegator = "0.9.1"
```

## Documentation

API documentation is available on [docs.rs/ark-delegator](https://docs.rs/ark-delegator).
